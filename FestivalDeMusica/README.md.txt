# Festival Música

Este proyecto es una página web dedicada a un festival de música.  
Aquí podrás encontrar información sobre el evento, artistas, horarios y más.

## Tecnologías usadas

- HTML puro
- CSS

## Cómo usar

1. Clona el repositorio:
git clone https://github.com/Enrique-Montes/Festival-Musica.git

2. Abre el archivo `index.html` en tu navegador para ver la página web.
